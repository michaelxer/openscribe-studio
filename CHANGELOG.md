# Changelog

## 0.2.1 - 2026-05-26

- Added release automation that publishes Windows and macOS zip assets to GitHub Releases.

## 0.2.0 - 2026-05-26

- Rebranded the project as OpenScribe Studio.
- Added a larger, more reliable desktop layout so Activity stays visible.
- Added job progress with percentage, elapsed time, ETA, and current phase.
- Added Cancel support for long-running jobs.
- Added user-facing documentation for model and performance choices.
- Added macOS build script and open-source project files.

## 0.1.0 - 2026-05-25

- Initial desktop app for local files, folders, and YouTube/video URLs.
- Added Markdown, SRT, VTT, TXT, and JSON transcript outputs.
- Added portable Windows build script.
